## Rules

- Components must be small and reusable
- One component = one responsibility
- Prefer composition over inheritance
- Props must be typed explicitly

## UI

- Prefer shadcn/ui
- Customization must be done by composition
- Do not duplicate existing components

## Prohibitions

- No API calls
- No React Query
- No form validation
